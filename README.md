# AI StudyBuddy Chatbot

## Project Overview
AI StudyBuddy is a simple educational chatbot developed to support SDG 4: Quality Education. It helps students learn basic concepts of AI, Python, Machine Learning, and Data Science.

## Problem Statement
Many students do not have access to personalized learning support. This project provides quick educational assistance through a chatbot.

## Proposed Solution
The chatbot accepts user questions and returns educational answers based on predefined knowledge topics.

## Features
- Student-friendly chatbot
- Answers AI-related questions
- Question history
- Simple web interface using Streamlit

## Technology Stack
- Python
- Streamlit
- GitHub

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   streamlit run AI_agent.py
